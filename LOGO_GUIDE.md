# CalRoute Logo Guide

## Logo Concept

The CalRoute logo combines a **calendar grid** (representing scheduling) with **routing dots and connecting lines** (representing team routing and connections). This represents the core value: smart scheduling that automatically routes meetings to the right team member.

## Logo Files

### 1. **Logo with Text** (`logo-with-text.svg`)
- **Use for:** Main branding, header logos, website hero
- **Dimensions:** Horizontal layout with icon + "CalRoute" text
- **Best for:** Desktop viewing, large displays

### 2. **Square Logo / Icon** (`favicon-logo.svg`)
- **Use for:** Favicon, app icon, small spaces
- **Dimensions:** Square format
- **Best for:** Browser tabs, app icons, social media avatars (500x500px)

### 3. **Wordmark** (`wordmark.svg`)
- **Use for:** Text-only branding, simple text header
- **Dimensions:** Text only, no icon
- **Best for:** Footer, navigation, text headers

### 4. **Icon Only** (`logo.svg`)
- **Use for:** Standalone icon, mark only
- **Dimensions:** Square icon
- **Best for:** Social links, small UI elements

## Color Specifications

- **Primary Color:** `#0D7377` (Teal)
- **Dark Variant:** `#0a5f63` (Darker Teal)
- **Gradient:** From #0D7377 to #0a5f63
- **White:** `#FFFFFF` (for icon elements)
- **Text:** `#1a1a1a` (Dark gray/black)

## Usage Guidelines

### Desktop / Web
- Use `logo-with-text.svg` for main headers
- Minimum size: 200px wide
- Maintain clear space (20% of logo height)

### Mobile / Small Spaces
- Use `favicon-logo.svg` (square icon)
- Minimum size: 48x48px
- Works as app icon, favicon, avatar

### Favicon Implementation
```html
<link rel="icon" type="image/svg+xml" href="/favicon-logo.svg" />
```

### Social Media
- 1200x630px for social cards (use logo-with-text.svg, scale)
- 500x500px for profile image (use favicon-logo.svg)
- 192x192px for app icons

## Brand Colors

```css
--primary: #0D7377;
--primary-dark: #0a5f63;
--background: #F7F4EF;
--text-dark: #1a1a1a;
--text-light: #1a1a1a/40;
```

## Logo Variations

- ✅ Full color (on white or light background)
- ✅ Icon only (for small spaces)
- ✅ Text only (for simple headers)
- ❌ Do NOT invert colors (grayscale not recommended)
- ❌ Do NOT stretch or distort
